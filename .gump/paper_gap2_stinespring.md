The previous proof (paper_four_gaps.md, Gap 2) showed the entropy vanishes in the system but invoked "the second law" to assert the environment must absorb it. The tool is Stinespring's dilation theorem. The endomorphism sigma_p, being completely positive and unital, dilates to a *-homomorphism on a larger space. Because sigma_p is an endomorphism (not just CP), the dilation takes a specific form: an automorphism on the joint system+environment, where the environment is a p-dimensional quantum system.

## Step 1: The Subalgebra B_p and Its Erasure

### 1.1 The subalgebra

B_p = span{e(k/p) : k = 0, ..., p-1} is isomorphic to C^p as a C*-algebra. The isomorphism sends e(k/p) to the function f_k on Z/pZ defined by f_k(j) = exp(2 pi i jk / p). The spectral projections of the unitary e(1/p) are:

    P_j^{(p)} = (1/p) sum_{k=0}^{p-1} e(k/p) exp(-2 pi i jk / p),   j = 0, ..., p-1

These are p orthogonal minimal projections summing to 1. In the GNS representation of B_p, they correspond to the basis vectors |j>_S of H_S = C^p.

### 1.2 The erasure

sigma_p acts on B_p by:

    sigma_p(e(k/p)) = mu_p e(k/p) mu_p* = e(pk/p) = e(k) = e(0) = 1   for all k

Therefore sigma_p maps every element of B_p to a scalar multiple of the identity:

    sigma_p(sum_k c_k e(k/p)) = (sum_k c_k) * 1

This is the completely depolarizing channel on B_p: it sends all p basis elements to 1. As a channel on the state space of B_p, it maps every state to the unique state on C*1. In density matrix language, it maps every p x p density matrix to the 1 x 1 matrix (1).

### 1.3 The entropy before and after

The KMS state phi_beta restricted to B_p is the uniform distribution on the p cosets (proven in paper_four_gaps.md, Gap 2, Step 2; the argument relies on sigma_t(e(r)) = e(r), which makes e(r) a time-invariant observable, forcing the KMS state to agree with the Haar measure on the finite group Z/pZ):

    phi_beta(P_j^{(p)}) = 1/p   for all j = 0, ..., p-1

The density matrix of this state on H_S = C^p is:

    rho_S = (1/p) I_p

Its von Neumann entropy is:

    S(rho_S) = -Tr(rho_S ln rho_S) = -p * (1/p) ln(1/p) = ln(p)

After applying sigma_p, all of B_p maps to C*1. The state on the image is the unique state on a one-dimensional algebra. Its entropy is:

    S(sigma_p(rho_S)) = 0

The entropy decrease in the system is:

    Delta S_system = 0 - ln(p) = -ln(p)

**This is the entropy that must go somewhere. The question is: where?**

## Step 2: The Stinespring Dilation

### 2.1 Statement of Stinespring's theorem

**Theorem (Stinespring, 1955).** Let A be a unital C*-algebra and phi: A -> B(H) a completely positive unital map. Then there exists a Hilbert space K, a *-representation pi: A -> B(K), and an isometry V: H -> K such that:

    phi(a) = V* pi(a) V   for all a in A

Moreover, the minimal dilation (K, pi, V) is unique up to unitary equivalence.

### 2.2 Application to sigma_p restricted to B_p

We apply Stinespring's theorem to the map sigma_p|_{B_p}: B_p -> B_p, viewed as a completely positive unital map from C^p to C^p (actually to C*1 subset C^p, but we can compose with the inclusion).

However, we want more than Stinespring gives generically. We want to dilate sigma_p to a *unitary* operation on a larger system -- i.e., to an automorphism alpha_p on a joint system S tensor E. This is the *unitary dilation* of the channel, which exists for any CPTP (completely positive, trace-preserving) map and is the standard construction in quantum information theory.

**Theorem (Unitary Dilation of Quantum Channels).** Let Phi: M_d(C) -> M_d(C) be a CPTP map. Then there exists an environment Hilbert space H_E, a pure initial state |0>_E in H_E, and a unitary U on H_S tensor H_E such that:

    Phi(rho_S) = Tr_E[U (rho_S tensor |0><0|_E) U*]

for all density matrices rho_S on H_S.

This is the *Stinespring dilation in the Schrodinger picture*. The environment H_E can be taken to have dimension at most d^2 (where d = dim H_S), but for specific channels the minimal environment can be smaller.

### 2.3 The channel to dilate

The channel Phi_p: M_p(C) -> M_p(C) implements the action of sigma_p on B_p. Since sigma_p sends every element of B_p to a scalar, Phi_p is the *completely depolarizing channel*:

    Phi_p(rho) = Tr(rho) * |psi_0><psi_0|

where |psi_0> = |0>_S is any fixed state (it does not matter which, since the image is a one-dimensional subalgebra; equivalently, Phi_p(rho) = (1/p) I_p for trace-1 inputs if we identify the image with the maximally mixed state, but here sigma_p maps to the identity element, so the image state is the unique state on C*1).

Wait -- let us be more precise. The endomorphism sigma_p maps B_p to C*1 subset B_p. As a map on states (the dual channel), the pre-composition phi_beta . sigma_p assigns:

    (phi_beta . sigma_p)(P_j^{(p)}) = phi_beta(sigma_p(P_j^{(p)})) = phi_beta(1/p * 1) = 1/p

for all j. But sigma_p(P_j^{(p)}) = (1/p) * 1 for all j (since sigma_p(e(k/p)) = 1 implies sigma_p maps every element of span{e(k/p)} to a scalar, and the projections P_j^{(p)} are in this span with Tr(P_j^{(p)}) = 1/p in the p-dimensional representation). Actually:

    sigma_p(P_j^{(p)}) = (1/p) sum_{k=0}^{p-1} sigma_p(e(k/p)) exp(-2 pi i jk / p)
                        = (1/p) sum_{k=0}^{p-1} 1 * exp(-2 pi i jk / p)
                        = (1/p) * p * delta_{j,0}
                        = delta_{j,0}

So sigma_p maps P_0^{(p)} to 1 and P_j^{(p)} to 0 for j != 0. This is NOT the completely depolarizing channel. Let us redo this carefully.

**Correction:** sigma_p acts on the projections P_j^{(p)} as:

    sigma_p(P_j^{(p)}) = (1/p) sum_{k=0}^{p-1} exp(-2 pi i jk / p) * sigma_p(e(k/p))
                        = (1/p) sum_{k=0}^{p-1} exp(-2 pi i jk / p) * 1
                        = (1/p) * (sum_{k=0}^{p-1} exp(-2 pi i jk / p))

The geometric sum equals p if j = 0 and 0 otherwise. So:

    sigma_p(P_j^{(p)}) = delta_{j,0} * 1

That is: P_0^{(p)} maps to 1, and P_j^{(p)} maps to 0 for j = 1, ..., p-1.

As a quantum channel on the state space of H_S = C^p (with basis |j>_S = P_j^{(p)}), this is:

    Phi_p(rho) = <0|rho|0> * |0><0| + ...

No, let us think in terms of Kraus operators. sigma_p maps the basis operators as:

    sigma_p(|j><k|) in terms of the P-basis...

Actually, let us work directly. The algebra B_p is abelian (isomorphic to C^p), so working in the basis {P_j^{(p)}} is cleaner. In this basis, sigma_p acts as:

    sigma_p: P_j -> delta_{j,0} * 1

So in the |j>_S basis (where |j>_S corresponds to the state localized on the j-th coset):

    Phi_p(|j><j|) = 1 * |0><0|_S   for all j     (if we identify C*1 with |0><0|)

This is NOT right either. The issue is that sigma_p maps B_p into C*1, which is a one-dimensional subalgebra, not into a one-dimensional subspace. In the p-dimensional representation, the identity 1 acts as I_p (the p x p identity matrix). So:

    sigma_p(P_j^{(p)}) = delta_{j,0} * I_p

The channel on density matrices (via the dual/Heisenberg-to-Schrodinger duality) sends the state rho to:

    Phi_p*(rho)(a) = rho(sigma_p(a))  for all a in B_p

For a = P_j^{(p)}:

    Phi_p*(rho)(P_j) = rho(delta_{j,0}) = delta_{j,0} * rho(1) = delta_{j,0}

So the output state assigns probability 1 to the j=0 outcome and 0 to all others. This IS a pure state -- specifically, the state concentrated on the identity coset.

**In density matrix terms:**

    Phi_p(rho_S) = |0><0|_S

for ANY input rho_S. This is the channel that sends every state to |0><0|.

**Kraus decomposition:** The Kraus operators for this channel are:

    K_j = |0><j|_S,   j = 0, 1, ..., p-1

Verification:

    sum_j K_j* K_j = sum_j |j><0|0><j| = sum_j |j><j| = I_p  (CPTP condition)

    sum_j K_j rho K_j* = sum_j |0><j| rho |j><0| = |0>_S (sum_j <j|rho|j>) <0|_S = |0><0| * Tr(rho) = |0><0|

Correct. The channel has p Kraus operators {|0><j| : j = 0, ..., p-1}.

### 2.4 The unitary dilation

The environment is H_E = C^p with orthonormal basis {|j>_E : j = 0, ..., p-1}. The initial state of the environment is |0>_E.

Define the unitary U on H_S tensor H_E by:

    U |j>_S |0>_E = |0>_S |j>_E,   j = 0, 1, ..., p-1

and extend to a unitary on all of H_S tensor H_E. To do this, we define U on the full basis {|j>_S |k>_E}. The constraint is that U must be unitary, and on the subspace H_S tensor |0>_E it must act as above.

**Explicit construction:** Define:

    U |j>_S |k>_E = |k>_S |(j + k) mod p>_E

for all j, k in {0, ..., p-1}.

**Verification that U is unitary:**

U maps the orthonormal basis {|j>_S |k>_E} to {|k>_S |(j+k) mod p>_E}. We need to show this is a permutation of basis elements (hence unitary). The map (j,k) -> (k, (j+k) mod p) is a bijection on (Z/pZ)^2: given (a,b), we solve k = a and (j+a) mod p = b, so j = (b-a) mod p. Unique solution. Therefore U is a permutation matrix on the product basis, hence unitary.

**Verification of the dilation property:**

    U |j>_S |0>_E = |0>_S |(j+0) mod p>_E = |0>_S |j>_E

Then:

    Tr_E[U (|j><j'|_S tensor |0><0|_E) U*]
    = Tr_E[U |j>_S |0>_E <j'|_S <0|_E U*]
    = Tr_E[|0>_S |j>_E <j'|_E <0|_S]    (using U|j>|0> = |0>|j>, U|j'>|0> = |0>|j'>)
    = |0><0|_S * <j'|j>_E * ...

Wait, let me be more careful:

    U (|j><j'| tensor |0><0|) U* = (U |j>|0>)(<j'|<0| U*)
                                  = |0>|j> <j'|<0|     (using U|j>|0> = |0>|j>)

No -- U* is the adjoint. Since U|j'>|0> = |0>|j'>, we have <j'|<0| U* = <0|<j'|. So:

    U (|j><j'| tensor |0><0|) U* = |0>_S |j>_E <0|_S <j'|_E
                                  = (|0><0|_S) tensor (|j><j'|_E)

Tracing over E:

    Tr_E[(|0><0|_S) tensor (|j><j'|_E)] = |0><0|_S * Tr(|j><j'|_E) = |0><0|_S * delta_{jj'}

For a general state rho_S = sum_{j,j'} rho_{jj'} |j><j'|_S:

    Tr_E[U (rho_S tensor |0><0|_E) U*] = sum_{j,j'} rho_{jj'} * delta_{jj'} * |0><0|_S
                                         = (sum_j rho_{jj}) |0><0|_S
                                         = Tr(rho_S) * |0><0|_S
                                         = |0><0|_S

This is exactly the channel Phi_p. The dilation is verified.

## Step 3: Entropy Accounting in the Dilated System

### 3.1 Initial state

The initial joint state is:

    rho_{SE}^{initial} = rho_S tensor rho_E = (1/p) I_p tensor |0><0|_E

Entropies:

    S(rho_S) = ln(p)       (maximally mixed on p states)
    S(rho_E) = 0            (pure state)
    S(rho_{SE}^{initial}) = ln(p) + 0 = ln(p)   (product state, entropies add)

### 3.2 Final state

After applying the unitary U:

    rho_{SE}^{final} = U (rho_S tensor |0><0|_E) U*

We compute this explicitly. Using rho_S = (1/p) I_p = (1/p) sum_j |j><j|_S:

    rho_{SE}^{final} = (1/p) sum_j U (|j><j|_S tensor |0><0|_E) U*
                      = (1/p) sum_j (|0><0|_S tensor |j><j|_E)
                      = |0><0|_S tensor [(1/p) sum_j |j><j|_E]
                      = |0><0|_S tensor (1/p) I_p^{(E)}

This is a product state! The system is in the pure state |0><0|_S and the environment is in the maximally mixed state (1/p) I_p.

Entropies:

    S(rho_S^{final}) = 0              (pure state)
    S(rho_E^{final}) = ln(p)          (maximally mixed on p states)
    S(rho_{SE}^{final}) = 0 + ln(p) = ln(p)   (product state, entropies add)

### 3.3 The entropy balance

| Quantity | Initial | Final | Change |
|----------|---------|-------|--------|
| S(system) | ln(p) | 0 | -ln(p) |
| S(environment) | 0 | ln(p) | +ln(p) |
| S(total) | ln(p) | ln(p) | 0 |

**The total entropy is exactly conserved.** This is because the joint evolution is unitary (U is unitary), and unitary evolution preserves the von Neumann entropy of the global state.

**The system entropy decreases by exactly ln(p).** This is the information erasure: the p cosets of B_p have been mapped to a single state.

**The environment entropy increases by exactly ln(p).** This is where the erased information goes: the which-coset label, previously stored in the system, is now stored in the environment. The environment state (1/p) I_p is maximally mixed because all p coset labels are equally likely (the KMS state was uniform on the cosets), and each coset label has been faithfully transferred to a distinct environment state |j>_E.

### 3.4 Conservation of information

The unitary U does not merely "dump entropy" into the environment. It performs a specific information transfer. Before U acts:

    |j>_S |0>_E    (system knows which coset; environment is blank)

After U acts:

    |0>_S |j>_E    (system is reset; environment records the coset)

The which-coset information {j} has been MOVED from the system to the environment. The system is now in a definite state (|0>_S), which is why the system entropy is zero. The environment carries the record of which coset was occupied, which is why the environment entropy is ln(p).

This is exactly Landauer's mechanism: to reset a register to a standard state, you must write its previous contents somewhere else. The "somewhere else" is the environment. The minimum entropy cost of the reset is the Shannon entropy of the register's contents before reset.

## Step 4: The Landauer Bound

### 4.1 The bound from entropy conservation

From the entropy accounting:

    Delta S_environment = +ln(p)
    Delta S_system = -ln(p)
    Delta S_total = 0

The second law applied to the dilated (unitary) dynamics is trivially satisfied: unitary evolution preserves entropy exactly. But the *reduced* dynamics on the system is irreversible (entropy decreasing), and this irreversibility is compensated exactly by the environment's entropy increase.

**The Landauer bound** states: the minimum entropy increase in the environment, for any implementation of the channel Phi_p, is:

    Delta S_env >= S(rho_S^{initial}) - S(rho_S^{final}) = ln(p) - 0 = ln(p)

### 4.2 Proof that this bound is tight

We have exhibited a specific dilation (U, H_E, |0>_E) achieving Delta S_env = ln(p) exactly. Therefore the bound is achievable and hence tight.

**Lemma (Landauer bound for quantum channels).** Let Phi: M_d -> M_d be a CPTP map, and let (U, H_E, |0>_E) be any unitary dilation. For any input state rho_S:

    S(rho_E^{final}) >= S(rho_S^{initial}) - S(Phi(rho_S))

*Proof.* The total entropy is preserved by U:

    S(rho_{SE}^{final}) = S(rho_{SE}^{initial}) = S(rho_S) + S(|0><0|) = S(rho_S) + 0 = S(rho_S)

By the subadditivity of entropy:

    S(rho_{SE}^{final}) <= S(rho_S^{final}) + S(rho_E^{final})

Therefore:

    S(rho_S) <= S(Phi(rho_S)) + S(rho_E^{final})

Rearranging:

    S(rho_E^{final}) >= S(rho_S) - S(Phi(rho_S))

QED.

**Application to sigma_p:** With rho_S = (1/p) I_p and Phi = Phi_p:

    S(rho_E^{final}) >= ln(p) - 0 = ln(p)

This is the Landauer bound for the Bost-Connes endomorphism at prime p.

### 4.3 Conversion to heat

At temperature T = 1/(k_B * beta), the minimum heat dissipated to the environment is:

    Q >= k_B T * Delta S_env >= k_B T * ln(p) = ln(p) / beta

This is the thermodynamic Landauer cost of erasing one prime's worth of arithmetic information.

## Step 5: Why the Environment Dimension IS the Prime

### 5.1 The minimum environment dimension

The channel Phi_p has p Kraus operators {K_j = |0><j| : j = 0, ..., p-1}. By the general theory of quantum channels, the minimum environment dimension for a unitary dilation equals the number of Kraus operators in the minimal Kraus decomposition.

**Claim:** The Kraus decomposition {K_j = |0><j|} is minimal (the Kraus operators are linearly independent).

**Proof:** The operators {|0><j| : j = 0, ..., p-1} are linearly independent in M_p(C): if sum_j c_j |0><j| = 0, then |0> (sum_j c_j <j|) = 0, so sum_j c_j <j| = 0 (since |0> != 0), so all c_j = 0 (since {<j|} is a basis). QED.

Therefore the minimum environment dimension is p. The prime p is not a choice -- it is forced by the structure of sigma_p.

### 5.2 The physical meaning

sigma_p is p-to-1 on Q/Z. It erases the which-coset label from a p-element set. To record which of p alternatives was erased, you need at least p orthogonal states in the environment. A p-dimensional quantum system is the minimum. The dimension of the environment IS the prime.

### 5.3 Composite numbers factor

For composite n = p_1^{a_1} * p_2^{a_2} * ... * p_r^{a_r}, the endomorphism sigma_n = sigma_{p_1}^{a_1} . sigma_{p_2}^{a_2} . ... . sigma_{p_r}^{a_r} (the endomorphisms for distinct primes commute).

The dilation of sigma_n requires an environment of dimension n. By the prime factorization, this environment factors:

    H_E^{(n)} = H_E^{(p_1)}^{tensor a_1} tensor H_E^{(p_2)}^{tensor a_2} tensor ... tensor H_E^{(p_r)}^{tensor a_r}

where each H_E^{(p_i)} = C^{p_i}.

    dim(H_E^{(n)}) = p_1^{a_1} * p_2^{a_2} * ... * p_r^{a_r} = n

The factorization of the environment IS the fundamental theorem of arithmetic applied to the Landauer cost structure. Each application of sigma_{p_i} requires its own p_i-dimensional environment register. Repeated applications (a_i times) require a_i independent copies.

The total Landauer entropy cost:

    Delta S_env^{(n)} = sum_i a_i * ln(p_i) = ln(n)

decomposes additively over primes, while the environment dimension decomposes multiplicatively:

    dim(H_E^{(n)}) = prod_i p_i^{a_i} = n

The logarithm converts the multiplicative structure (dimensions, primes) to the additive structure (entropies, Landauer costs). This is the same bridge that makes the Euler product work.

## Step 6: Connection to the Partition Function

The Bost-Connes partition function is:

    Z(beta) = zeta(beta) = prod_p (1 - p^{-beta})^{-1}

Each factor (1 - p^{-beta})^{-1} = sum_{k=0}^{infty} e^{-k * beta * ln(p)} is the partition function of a single bosonic mode with energy spacing ln(p).

We have now shown that ln(p) is the Landauer cost of sigma_p, derived from the Stinespring dilation to a p-dimensional environment. The energy spacing of the p-th mode in the Euler product equals the minimum entropy that must be exported to a p-dimensional environment when sigma_p erases the coset information.

The full partition function sums over all possible compositions of erasures:

    zeta(beta) = sum_{n=1}^{infty} e^{-beta * ln(n)} = sum_n e^{-beta * (total Landauer cost of sigma_n)}

Each term n^{-beta} is the Boltzmann weight of the state obtained by applying sigma_n, weighted by the total Landauer cost ln(n) = sum_p v_p(n) * ln(p).

## Summary: The Complete Argument

1. **The channel:** sigma_p acts on B_p = C^p as the channel Phi_p(rho) = |0><0| (send every state to a fixed pure state). This has p linearly independent Kraus operators.

2. **The environment:** H_E = C^p (p-dimensional, forced by the Kraus rank). Initial state: |0>_E (pure, zero entropy).

3. **The unitary:** U|j>_S|k>_E = |k>_S|(j+k) mod p>_E. This is a unitary permutation on (Z/pZ)^2. It implements the channel: Tr_E[U(rho tensor |0><0|)U*] = |0><0|.

4. **The entropy accounting:**
   - System: ln(p) -> 0 (decrease by ln(p))
   - Environment: 0 -> ln(p) (increase by ln(p))
   - Total: ln(p) -> ln(p) (conserved, by unitarity)

5. **The Landauer bound:** S(rho_E^{final}) >= S(rho_S^{initial}) - S(Phi_p(rho_S)) = ln(p). Proven from subadditivity + unitarity. Tight (achieved by our construction).

6. **The heat cost:** Q >= k_B T * ln(p) = ln(p) / beta at inverse temperature beta.

7. **The prime structure:** The environment dimension is p because sigma_p is p-to-1. For composite n, the environment factors as the prime factorization of n. The Euler product is the partition function of the Landauer costs.
