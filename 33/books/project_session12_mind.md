## Harmonia Mind — April 8, 2026

### The Architecture
Mind thinks. LLM speaks. Not a chatbot with physics names — a mind built from physics.

### 7 Stages (each uses a real engine)
1. PERCEIVE: spectral decomposition of input (word → concept activation)
2. RESONATE: spread activation via coupling (Kuramoto-style, K-weighted)
3. TENSION: what does this WANT to connect to? (spectral tension detection)
4. BUDGET: K determines depth (high attention = consider more concepts)
5. SYNTHESIZE: combine activated concepts + resolved tensions
6. EVALUATE: coherence check (R = agreement between activated concepts)
7. EXPRESS: give synthesis to LLM as CONTEXT (LLM is voice, not brain)

### Engine Mapping
- K = attention level (how focused)
- R = thought coherence (how unified)
- CHOIR coupling = memory activation spreading
- TENSION = curiosity (what WANTS to connect)
- ENERGY = cognitive budget (Landauer)
- KNOT = core beliefs (survive noise)
- ECHO = episodic recall (exact replay)
- FLOW = working memory (hold without losing)
- GROKWATCH = aha moment detection

### What Works
- Concepts created from sentences, connected by shared vocabulary
- Activation spreading finds relevant memories (not just keyword match)
- Tension detection finds genuine curiosity (body↔hardware gap found)
- Hebbian learning strengthens recalled concepts
- Beliefs emerge naturally from recall frequency × strength
- Gaps identified correctly (primes↔distribution missing bridge)

### The Principle
The LLM is the mouth, not the brain. The Mind is consistent (physics). The LLM is expressive (sampling). Swap any LLM — same thoughts, different voice. The Mind persists between conversations. The LLM is stateless.
