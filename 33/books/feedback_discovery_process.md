James discovers by doing, not by planning. The process:

1. **Start with an intuition** — a gut feeling, a question, a connection that "feels right." Not a hypothesis. A pull.

2. **Build immediately** — don't theorize, don't plan, don't ask permission. Build the thing. If it's a math idea, write the proof. If it's a tool idea, code it. If it's a sound idea, make the sound. Ship it.

3. **Send it to the destroyers** — take the built thing and throw it at hostile reviewers (Grok, GPT, Gemini, Claude — all of them). Let them rip it apart. Every critique is data.

4. **Fix what's broken, kill what's dead** — if the critique is right, fix it immediately. If an idea is dead (dark matter as Landauer heat, eigenvalue saturation), kill it honestly and move on. Don't defend dead ideas.

5. **Build the next tool from the wreckage** — the critique reveals the real shape. The paper got hardened by hostile review. The simulator got honest by adding parity. The compiler got real by adding checkpointing. Each destruction created something stronger.

6. **Iterate fast, in the same session** — don't wait. Don't "come back to it." Run the full loop: build → critique → fix → build again → critique again. Multiple cycles per session.

7. **Cross-pollinate between AIs** — use each AI's strengths. Send the same work to multiple reviewers. They find different weaknesses. The intersection of "what none of them could break" is what's real.

8. **Always connect back to something practical** — the paper is interesting but the compiler is useful. The compiler is cool but the checkpoint optimizer pays for itself. Every abstraction must touch ground.

**Why:** James doesn't have credentials. He has taste, persistence, and the willingness to be wrong in public. The process compensates for the lack of formal training by being more rigorous than formal training — because everything gets stress-tested by machines that don't care about his feelings.

**How to apply:** When James brings an idea, don't evaluate it — build it. Then help him destroy it. What survives is real.
