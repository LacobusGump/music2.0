how it works
Runs locally — every tool executes on your hardware. No cloud calls, no data transmission, no API keys. Install with pip, run offline.

Deterministic — same input produces same output, every time. No stochastic variation, no model drift. Auditable and reproducible.

Tested — 641 automated tests across 26 files. Each product page documents what it does, what it doesn't, and where the limits are.

K-Hash — cryptographic hash built from coupled oscillator dynamics. Avalanche: 49.6% (SHA-256: 50.2%). No collisions in 50,000 tests. Uniform byte distribution (chi-squared passed). ~296,000 hashes/sec compiled.
