# GUMP-1: Reactive Transduction Engine

**Information-to-Material Silicon Synthesis**

---

## Executive Summary

GUMP-1 is an experimental ASIC designed to bridge high-level generative software and physical silicon. Using a Pareto-optimal activation checkpoint schedule derived from an entropy-aware compiler, the architecture selectively toggles between Analog ReRAM Crossbars and Digital Pipelined Logic to maintain structural stability under computational stress.

The chip physically embodies the entropy structure of its generative model: compute-intensive zones map to systolic arrays, memory-heavy zones map to distributed SRAM, and the floorplan is a 1:1 reflection of the model's dependency graph.

## The Core Innovation

Traditional hardware forces algorithms into a static grid. GUMP-1 uses the algorithm's own entropy — derived from a [Checkpoint Optimizer](https://begump.com/tools/checkpoint.html) — to dictate the physical floorplan.

- **Logic as Topology:** Compute-intensive zones mapped to Systolic Arrays to spread thermal density.
- **Memory as Buffer:** Stored activations placed in Distributed SRAM as physical heat sinks for recompute logic.
- **Material Limits:** Design respects the Landauer Limit (E = kT·ln2), operating at the edge of physical possibility.

## The Pareto Switch

The core logic is defined by a software-derived schedule that toggles between:
- **Analog ReRAM Crossbars** — zero-movement computation (weight storage + multiply-accumulate)
- **Digital SRAM** — state persistence (activation checkpoints)

The schedule is generated by the [Entropy-Aware Compiler](https://begump.com/tools/compiler.html), which classifies the computation's phase (GAS/LIQUID/CRYSTAL) and determines the optimal checkpoint placement.

## Phase-Driven Architecture

The compiler's dimensionless curvature measure **R** determines the chip's operating mode:

| R | Phase | Chip Behavior |
|---|-------|--------------|
| R < 1 | GAS | Store early layers in SRAM, recompute rest in crossbar |
| R ≈ 1 | LIQUID | Distribute checkpoints uniformly across BRAM |
| R > 1 | CRYSTAL | Minimal checkpointing, full compute-through |

This is validated: R predicts the cost of being wrong at **-0.799 correlation** across 5 real neural architectures (ResNet, GPT-2, BERT, U-Net, VGG).
