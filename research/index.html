HOW WE TEST — MM12P
0. Discriminate — correlation or coupling? No Landauer receipt, no point continuing.
1. Precise claim — state exactly what you're testing.
2. Disproof test — build the test that would kill it.
3. Ground truth — run against known, published data.
4. Adversarial — feed it the worst possible input.
5. Edge cases — boundary conditions, empty inputs, extremes.
6. Opposite claim — test whether the reverse is also true.
7. Ablation — remove the feature. Does anything change?
8. Next best — compare to the best existing alternative.
9. Regression — re-run ALL previously passing tests. New work must not break old results. The suite grows every version.
10. Verdict + alternative path — if 0–9 pass: proved, added to regression suite. If any step does not hold: state what, why, and where to go next. Never a dead end — always a direction.
