The Network — STRING-Sourced

The previous version of this page used a 90-node hand-built toy network with 19 manually selected crosstalk edges. The 10P audit flagged this: hub rankings were artifacts of edge selection, not biology. CTNNB1 was #1 because we gave it 5 crosstalk edges by hand.

This version uses real protein-protein interaction data from the STRING database (v12, Homo sapiens, taxon 9606). We queried 10 major cancer driver proteins at combined confidence score ≥ 700 (high confidence). STRING returned 37 verified interactions among these 10 proteins.

82% completeness means these 10 proteins are deeply interconnected. This is expected: they are among the most-studied cancer drivers in the human proteome. The high interconnection is biology, not an artifact of our edge selection.

Cascade Model

The cascade principle still holds: knocking out a high-K hub disrupts more downstream signaling. But now the hub ranking is grounded in real interaction data. TP53 knockout should cascade the furthest because it has the most connections — and indeed, TP53 loss is the single most common event in human cancer, occurring in ~50% of all cancers.

Cascade Predictions (STRING-Informed)

TP53 knockout: connects to all 9 other drivers. Loss cascades through every major cancer pathway.
AKT1 knockout: 1,820 partners. Central node in PI3K/AKT/mTOR survival signaling.
MYC knockout: 1,579 partners. Master transcription factor — amplified in ~30% of all cancers.
PTEN loss: 995 partners, but connects to all 9 drivers locally. Unleashes PI3K/AKT pathway.
RB1 knockout: only 381 partners, 3/9 local edges. More isolated — loss primarily affects cell cycle.

Method

STRING database v12 API query for 10 cancer driver proteins (Homo sapiens, taxon 9606). Combined confidence score threshold: 700 (high confidence). Total interaction partner counts from STRING interaction_partners endpoint (all confidence levels). K computed two ways: global (total partners / max partners) and local (edges within 10-protein subnetwork / 9). No hand-selected edges. No machine learning.

Honest limits: this is a 10-protein subnetwork of the full human interactome (~20,000 proteins). STRING interaction counts include all evidence channels (experimental, database, text-mining, co-expression, genomic context). Text-mining dominates for well-studied cancer genes, which inflates counts for famous proteins like TP53. The local network at this threshold is 82% complete, meaning most of these proteins interact with most others — which is biologically real but means local degree is less discriminating than global partner count. STRING data, not hand-curated edges.

Data retrieved from STRING API, April 2026. Computation: API query + degree counting. Pathway data: STRING v12 (string-db.org).
